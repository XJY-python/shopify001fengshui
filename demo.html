功能演示

五行元素产品展示插件提供了以下核心功能：

  • 悬停显示产品分类下拉菜单
  • 点击分类标签筛选对应产品
  • 平滑的动画过渡效果
  • 响应式布局设计
  • Shopify主题集成支持
// 简化的筛选逻辑示例
function filterProducts(element, category) {
  const productCards = document.querySelectorAll(`#${element}-products .product-card`);
  productCards.forEach(card => {
    const cardCategory = card.getAttribute('data-category');
    card.style.display = cardCategory === category ? 'block' : 'none';
  });
}

安装指南

  1. 将 five-elements-display.liquid 文件复制到 Shopify 主题的 sections 目录
  2. 在主题设置中添加五行元素产品展示区块
  3. 配置各元素对应的产品集合
  4. 为产品添加相应的 data-category 属性
  5. 保存并发布主题更改
